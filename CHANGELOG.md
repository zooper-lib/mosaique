# Changelog

All notable changes to this project will be documented in this file.

## [0.1.0] - 2025-11-26

### Added
- Initial release of Mosaique
- `Region` widget for defining placeholders in shell layouts
- `MosaiqueShellRoute` for creating shell layouts with named regions
- `MosaiqueViewRoute` for injecting views into specific regions
- Support for fixed regions (persistent UI elements)
- Support for nested shells (shells within shells)
- Automatic AnimatedSwitcher transitions between views
- Full integration with go_router
- Example app demonstrating all features

### Features
- Declarative route-driven architecture
- Type-safe route parameters
- Infinite shell nesting support
- Customizable transition animations per region
- Simple 3-class API
